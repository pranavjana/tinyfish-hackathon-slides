What we heard

The good

Integration
"The API is cool" — very good integration experience across the board
Product
Live preview is really eye-catching. People were drawn to it immediately.
Onboarding
Quick to get started — people went from signup to running their first task within minutes.
10
What we heard

The hard truths

Perception
People still assume we're scrapers or an out-of-the-box solution for every website. Had to correct this repeatedly.
Scaling
Didn't manage to scale some services in time. Users got 2x concurrency instead of the 10x we promised. Nothing broke — just slow.
Feature requests
Browser session persistence, faster agent speed, more reliable anti-bot solutions.
