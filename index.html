THE BASICS

What is
an Agent?

A system that takes actions on your behalf based on goals, not instructions — often with a level of reasoning involved.

SCRIPT

"Send an email to john@example.com."

One input, one output. Done.

AGENT

"Get me to Tokyo next week as cheap as possible."

Searches multiple airlines. Compares prices and layovers. Checks your calendar for conflicts. Books the best option. Adds it to your itinerary.

06 / 14
ACTION LAYER

Tools.

  • Tools = functions the agent can call
  • Each tool does one thing: send email, query DB, create event
  • The model doesn't execute tools — it requests them
const result = await generateText({
  model: openai('gpt-4o'),                    // which LLM to use
  tools: { weather: tool({
    description: 'Get weather in a city',      // tells the model what this tool does
    parameters: z.object({ city: z.string() }), // what inputs the tool accepts
    execute: async ({ city }) => getWeather(city), // function you define for the agent to run
  }) },
});

Tool definition in the Vercel AI SDK — we'll use this later!

THE FLOW

How Tool Calling
Works.

🤖
"What's the weather
in Singapore?"
→
CALLS TOOL
getWeather("SG")
→
GETS DATA BACK
the agent reads this info
{ temp: 31, humid: 82% }
→
🤖
"It's 31°C and
humid in Singapore!"
  • Orchestration layer (Vercel AI SDK, Claude Code, LangChain, etc.) checks permissions & validates parameters
  • Executes the function, captures the result
  • Feeds structured data back — agent reasons about what to do next
  • The agent knows what the tool does, not how

PERSISTENCE

Context &
Memory.

Even with 1M token windows, reasoning degrades past ~200K. Context rot is real.

WHAT'S IN YOUR MEMORY
  • System prompt
  • Tool definitions
  • Skills / .md files
  • Conversation history

Less bloat in memory = better agent. Find the balance.

CHOOSING RIGHT

Model
Selection.

OPENAI
GPT-5 mini (high)
Agentic index: 45. Strong tool calling, good reasoning.
ANTHROPIC
Claude 4.5 Haiku
Agentic index: 40. Fast, reliable, great for most agent tasks.
GOOGLE
Gemini 3.1 Flash Lite
Agentic index: 26. Cheapest, fastest. Good for simple tasks.

Not every task needs the most powerful model. Different models have different agentic capabilities — a model that isn't the "smartest" may have better tool calling and agentic performance.
